# 🎢 減衰付き単振動シミュレーション（C言語）

減衰項を含む単振動の運動の数値シミュレーションを行う．
- **オイラー法** を使って時間発展をシミュレートします。

- 初期条件とパラメータを指定してx(t),v(t)の時間発展を見る
- 減衰の強さによる運動の変化を観察する．
- 出力を `.csv` にして Python 等で可視化

初期パラメータ

- 初期位置 \( x_0 \)
- 初期速度 \( v_0 \)
- 質量 \( m \)
- ばね定数 \( k \)
- 減衰係数 \( \gamma \)
- 時間ステップ \( \Delta t \)
- シミュレーション時間 \( T \)

### 📤 出力（標準出力 or ファイル）

- 各時刻 \( t \) における \( x(t) \), \( v(t) \)
